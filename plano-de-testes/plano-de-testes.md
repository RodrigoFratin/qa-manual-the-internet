# Plano de Testes

## 1. Objetivo
Verificar o funcionamento das principais funcionalidades do site
https://the-internet.herokuapp.com, garantindo que os recursos testados
funcionem conforme o esperado do ponto de vista do usuário final.

## 2. Sistema Testado
Site: https://the-internet.herokuapp.com

## 3. Escopo
Serão testadas as seguintes funcionalidades:
- Form Authentication (Login)
- Checkboxes
- Dropdown
- File Upload

## 4. Fora do Escopo
- Testes de performance
- Testes de segurança
- Testes em dispositivos móveis

## 5. Ambiente de Teste
- Sistema operacional: macOS
- Navegador: Google Chrome
- Tipo de teste: Manual

## 6. Estratégia de Teste
Serão realizados testes funcionais manuais, baseados em casos de teste
pré-definidos e testes exploratórios para identificação de falhas ou
comportamentos inesperados.
